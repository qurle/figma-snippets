# Plugin snippets for Figma

Just some small automation tasks made with Figma Plugin API.
See actual code at [src folder](https://github.com/qurle/figma-snippets/blob/main/src/)


## Snippets

| Snippet | Description |
|---------|-------------|
| applyFills | Create lots copies of element with differents fills applied to it |
| applyMinAndMax | Set minimal and maximum height and width of element equal to current width and height |
| bindSize | Bind width and height of elements to variables (if proper ones exist) |
| clone | Create clone of selection |
| convertColors | Color conversion utilities |
| fill | Set element to fill the parent by setting width and height |
| generateVariableValues | Get variables from fills of elements and print them as text |
| leaveOneImage | Remove multiple image fills |
| printNames | Copy layer names, divided by new lines |
| renameCodeSyntax | Rename code syntax of all the variables |
| renameProperties | Batch rename properties (rules are hardcoded) |
| swapTextStyleGroups | Replace text styles based on groups |
| wrapInAutoLayout | Make selected frames an autolayout, add top-level children frame inside |
