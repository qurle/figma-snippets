# Plugin snippets for Figma

Just some small automation tasks made with Figma Plugin API.
See actual code at [src folder](https://github.com/qurle/figma-snippets/blob/main/src/)


## Snippets

| Snippet | Description |
|---------|-------------|
| **Apply Fills** | Create lots copies of element with differents fills applied to it |
| **Apply Min And Max** | Set minimal and maximum height and width of element equal to current width and height |
| **Bind Size** | Bind width and height of elements to variables (if proper ones exist) |
| **Clone** | Create clone of selection |
| **Convert Colors** | Color conversion utilities |
| **Fill** | Set element to fill the parent by setting width and height |
| **Generate Variable Values** | Get variables from fills of elements and print them as text |
| **Leave One Image** | Remove multiple image fills |
| **Print Names** | Create a text layer with selected layers names, divided by new lines |
| **Rename Code Syntax** | Rename code syntax of all the variables |
| **Rename Properties** | Batch rename properties (rules are hardcoded) |
| **Swap Text Style Groups** | Replace text styles based on groups |
| **Wrap In Auto Layout** | Make selected frames an autolayout, add top-level children frame inside |
